# Study Planner – MERN Project
Erika Lankinen

## Description

Minimal MERN stack study planner app.

- Add, delete, and complete study tasks with subject and due date.
- Active / Completed lists and subject filter.
- Calendar view for daily tasks.

## Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/erikalank/SDS_Fullstack.git
    ```
2. Install backend:
  ```bash
    cd backend
    npm install
```
3. Install frontend:
 ```bash
  cd ../frontend
  npm install
```
4. Create .env in backend:
  ```bash
  MONGO_URI=your_mongodb_connection_string
  JWT_SECRET=your_secret_key
  PORT=5000
```
5. Start app:
  ```bash
  cd ..
  npm run dev
```
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
